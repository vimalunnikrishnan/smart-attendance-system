# Smart Attendance System (Flask)

## Developed by
BCA Final Year Student

## Description
This is a web-based Smart Attendance System developed using
Flask (Python) and SQLite. It allows faculty to manage students,
mark attendance, and generate attendance reports with percentage.

## Features
- Secure login with password hashing
- Add / Edit / Delete students
- Mark attendance (Present / Absent)
- Attendance percentage report
- Admin role-based access
- Mobile and desktop access

## Requirements
- Python 3.10 or above

## How to Run the Project

1. Open terminal inside project folder
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000
